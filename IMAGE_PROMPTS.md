# 30장 카드 일러스트 제작 기록

모든 그림은 Codex 내장 이미지 생성 도구로 이 프로젝트를 위해 새로 제작한 비공식 팬아트입니다. 공통 프롬프트 기준은 `세로 4:5 카드 일러스트, 캐릭터의 알려진 핵심 외형 유지, 글자·로고·워터마크·UI 없음, 8세 어린이용 비유혈 표현, 고품질 시네마틱 3D`입니다.

## DOORS 10장

- Seek: 검은 점액 인간형, 얼굴 중앙의 흰 눈 하나, 입과 의상 없음, 호텔 복도.
- Figure: 눈이 없고 갈비뼈가 드러난 붉은 괴물형 실루엣, 도서관.
- Rush: 검은 가시 연기 속의 일그러진 회백색 얼굴, 깊은 눈·코와 두 줄 이빨.
- Screech: 여러 촉수와 흰 눈을 가진 어두운 둥근 머리, 어두운 방.
- Eyes: 서로 다른 크기의 보라색 눈들이 모인 검은 연기 덩어리.
- Halt: 길쭉한 청록색 반투명 유령 인간형, 빛나는 눈과 검은 입, 푸른 복도.
- Ambush: 초록색 일그러진 얼굴과 커다란 검은 입, 반복 돌진 모션.
- Timothy: 열린 나무 서랍에서 튀어나오는 작은 검은 거미.
- Jack: 열린 옷장 속 검은 얼굴, 불균형한 흰 눈과 톱니 모양 미소.
- Guiding Light: 열쇠와 올바른 문을 연결하는 푸른 별빛 입자 궤적.

## SCP 10장

- SCP-173: 독자적으로 만든 땅콩형 콘크리트 조각, 철근, 검은 눈, 빨강·초록 페인트.
- SCP-096: 매우 길고 마른 회백색 인간형, 두 손으로 얼굴을 완전히 가린 자세.
- SCP-999: 친근한 주황색 젤리 생명체.
- SCP-049: 검은 로브와 새부리 의사 가면.
- SCP-131: 주황색과 노란색 눈알 포드 두 마리, 각각 큰 파란 눈 하나와 바퀴 받침.
- SCP-682: 거대한 장갑 비늘 악어형 파충류, 강화 격리실.
- SCP-3008-2: 노란 셔츠와 파란 바지, 얼굴이 없고 팔다리가 긴 매장 직원.
- SCP-2295: 여러 천 조각을 꿰맨 갈색 곰 인형, 의사 가방과 천으로 만든 심장.
- SCP-529: 뒷부분이 상처 없이 마법처럼 사라진 회색 줄무늬 고양이의 앞 절반.
- SCP-1762: 낡은 상자에서 날아오르는 여러 색의 종이접기 용.

## Brainrot 10장

- Tralalero Tralala: 파란 회색 상어, 정확히 세 다리와 파란 운동화 세 짝.
- Bombardiro Crocodilo: 악어 머리가 폭격기 동체에 직접 연결된 비행기 혼합체.
- Ballerina Cappuccina: 카푸치노 잔 머리와 발레리나 몸.
- Tung Tung Tung Sahur: 통나무 인간형, 긴 팔·다리, 어깨 위 나무 야구방망이.
- Cappuccino Assassino: 카푸치노 잔 머리, 검은 닌자 복장, 붉은 스카프와 연습봉.
- Lirili Larila: 코끼리 머리와 선인장 몸, 두 다리와 샌들, 등에 시계.
- Chimpanzini Bananini: 바나나 몸에 침팬지 얼굴·팔·다리가 결합된 혼합체.
- Bombombini Gusini: 거위 머리·깃털 몸과 제트기 날개·엔진이 결합된 혼합체.
- Brr Brr Patapim: 원숭이 얼굴, 이끼 낀 나무 몸, 잎사귀와 거대한 맨발.
- La Vaca Saturno Saturnita: 젖소 몸통에 토성형 행성과 고리가 결합된 우주 혼합체.

## 검수 규칙

- 30개 기본 카드 모두 `assets/`의 개별 WebP를 사용합니다. 원본 생성 PNG를 768×960, 품질 84의 WebP로 최적화해 모바일 로딩과 GitHub 업로드 용량을 줄였습니다.
- 토끼·오이·바나나 이모티콘 같은 임시 그림 카드는 카드 목록에서 제거했습니다.
- 기존에 저장된 임시 기본 카드는 앱 데이터 버전 5 마이그레이션에서 자동 제거하고, 이전 PNG 경로도 새 WebP 경로로 바꿉니다.
- 아이가 직접 만든 사진 카드와 진행도는 유지합니다.
- SCP-173은 특정 실물 조각 사진을 복제하지 않은 독자적인 콘크리트 조각 디자인입니다.
